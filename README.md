This project is a simple Python program that displays the message “Hello, World!” on the screen. It is commonly used as a beginner project to understand the basic structure of a program.


##  Aim of the Project
The aim of this project is to introduce basic programming concepts such as program execution and output display.


##  Objectives
- To write a simple Python program
- To display text output on the screen
- To understand how Python programs run


##  Scope of the Project
The project focuses only on displaying a message. It does not include user input, data storage, or advanced logic.


##  Tools and Technologies
- Programming Language: Python
- Code Editor: VS Code or any text editor
- Operating System: Windows, macOS, or Linux

When the program is executed, it prints the message “Hello, World!” to the screen.
